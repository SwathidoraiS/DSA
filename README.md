## 💡 My Approach
 
1. **Read** the problem carefully — understand inputs, outputs, constraints
2. **Think** before coding — brute force first, then optimize
3. **Code** with clean variable names and comments
4. **Analyze** time and space complexity
5. **Commit** — every solution gets pushed, no matter what
